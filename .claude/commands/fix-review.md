現在のブランチに紐づくPRのレビューコメントを取得し、整理して対応する項目を選択します。

## 目的

git-reviewスキルを簡単に呼び出し、GitHubのPRレビュー指摘への対応を効率化します。

## 実行内容

git-reviewスキルを呼び出して、以下の処理を自動化します:

- PRレビューコメントの取得と整理
- 対応項目の選択（AskUserQuestion）
- 選択された項目の修正実施
- コミット・プッシュ

詳細は `CLAUDE.md` の「🔧 スラッシュコマンド」セクションを参照してください。

## 使用方法

```bash
/fix-review
```

引数は不要です。現在のブランチから自動的にPRを検出します。

## 実行フロー

```
Skill: git-review

現在のブランチのPRレビューを取得してください。
```

## エラーケース

以下の場合はエラーが発生します:

- 現在のブランチに紐づくPRが存在しない
- gh CLI が未認証（`gh auth login` を実行してください）
- レビューコメントがまだ存在しない

詳細は git-review スキルのドキュメント（`.claude/skills/git-review/SKILL.md`）を参照してください。
